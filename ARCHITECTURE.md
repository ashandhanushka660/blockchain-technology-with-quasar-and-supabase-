## Technology Stack Details

### Frontend

- **Vue 3.5** - Progressive JavaScript framework
- **Composition API** - Modern Vue syntax
- **Quasar 2.18** - Material Design components
- **Tailwind CSS 4.1** - Utility-first CSS

### State Management

- **Pinia 3.0** - Vue store library
- **Persistent state** - LocalStorage integration

### Routing

- **Vue Router 4.6** - Official Vue router
- **Navigation guards** - Route protection
- **Lazy loading** - Code splitting

### Backend

- **Supabase** - Backend as a Service
- **PostgreSQL** - Relational database
- **Realtime** - WebSocket subscriptions
- **Auth** - JWT authentication

### Build & Deploy

- **Vite 7.3** - Next-gen build tool
- **PWA Plugin** - Progressive Web App
- **Vercel** - Deployment platform

### Additional Libraries

- **qrcode.vue** - QR code generation
- **@supabase/supabase-js** - Supabase client
- **sass-embedded** - SCSS support
